# 🧠 NLP Preprocessing Engine

This project is part of the Data Science Internship – February 2026 assignment.

## 📌 Task Overview
The goal of this project is to build a robust NLP preprocessing pipeline that can clean and transform raw, noisy text into meaningful structured data.

---

## 🚀 Features Implemented
- Text Lowercasing
- Removal of URLs and Emails
- Removal of Numbers
- Handling Repeated Characters (e.g., "soooo" → "soo")
- Removal of Special Characters & Emojis
- Tokenization
- Removal of Short Words (≤ 2 characters, except "no", "not")
- Extra Space Cleaning

---
